Contributing to open source projects is a fantastic way to learn, collaborate, and give back to the community. If you're new to open source, it can seem daunting at first, but fear not! This guide will walk you through making your first contribution using Git.

## Step 1: Find an Issue

Start by finding an issue in an open source project that interests you. You can browse platforms like GitHub, GitLab, or Bitbucket. Look for issues labeled as "good first issue" or "beginner-friendly" to find tasks suitable for newcomers.

## Step 2: Fork the Repository

Once you've found an issue, fork the repository to your own GitHub account. This creates a copy of the project that you can freely experiment with.

Click on the "Fork" button at the top right corner of the repository page on GitHub.
